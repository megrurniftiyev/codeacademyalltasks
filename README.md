# Code Academy - Full-Stack Development & Software Engineering Journey

This repository documents the technical knowledge, practical tasks, and the final project I completed during my intensive training at Code Academy. I successfully graduated from the program with a **High Honor** diploma.

## 🎓 Technical Skills & Curriculum

During the course, I mastered the fundamental and professional technologies required to build modern software solutions from scratch:

### 🔹 Backend & Database Engineering
* **C# & .NET / ASP.NET Core:** Proficient in Object-Oriented Programming (OOP), building scalable RESTful APIs, and managing server-side logic.
* **SQL:** Experienced in relational database design, writing complex queries, and data optimization.

### 🔹 Frontend Development
* **Web Fundamentals:** Creating responsive web interfaces using HTML5, CSS3, and JavaScript.
* **API Integration:** Bridging the gap between frontend and backend through effective API consumption.

## 🚀 Final Project: Mirai (Startup MVP)

The crowning achievement of my education is **Mirai**, an AI-integrated mobile application. This project represents the synthesis of my learning and is currently being developed as a startup.

* **Description:** An AI-powered mobile application featuring an integrated LLM (Large Language Model) chat system.
* **Tech Stack:**
    * **Mobile:** Flutter (Cross-platform)
    * **Backend:** Node.js
    * **Database:** Firebase (Real-time data management)
* **Status:** Developed as a startup MVP. For intellectual property reasons, the core source code is maintained in a private repository.

## 📂 Repository Structure

* **Mirai/:** Architectural structure and documentation for the final project.
